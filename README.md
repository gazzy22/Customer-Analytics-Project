# Customer Data Analytics Project

## Overview
This project demonstrates an end-to-end customer data analysis workflow, starting from raw Excel data to an interactive Power BI dashboard.

## Tools Used
- Excel (Data Cleaning & Transformation)
- Power BI (Visualization & Dashboarding)

## Project Structure
- Raw and cleaned datasets included
- Power BI dashboard file (.pbix)
- Dashboard screenshots for quick preview

## Key Insights
- Customer distribution by type and category
- Website and contact availability analysis
- Geographic customer concentration
